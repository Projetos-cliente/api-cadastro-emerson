# API DE CADASTRO

API com a funcionalidade de cadastrar clientes através de seu nome e CNPJ, também podendo consultá-los caso necessário.

# Serviços:

 /pessoas/cadastrar
 /pessoas/{identificador}
 /pessoas/buscarTodos


# Tecnologias Utilizadas:

versão do java: 8

versão do MySQL: 8.0.29
